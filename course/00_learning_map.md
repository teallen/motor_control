# 00 学习地图：从机械臂规划走向电机控制

你已有机械臂规控背景，所以学习电机控制时不需要先陷入功率器件细节。更高效的路径是先理解关节伺服系统如何把规划轨迹变成电机电流，再逐步补齐驱动器内部模型。

## 核心链路

```text
规划轨迹 q(t), dq(t), ddq(t)
  -> 逆动力学或前馈控制得到期望关节力矩 tau
  -> 通过减速比和力矩常数换算为电机电流 Iq
  -> 电流环和 FOC 控制三相电流
  -> 电机输出电磁力矩
  -> 减速器、连杆、负载产生实际运动
  -> 编码器反馈位置和速度
```

对规控算法工程师来说，最重要的是知道每一层的限制：

- 轨迹层：速度、加速度、jerk 和力矩是否连续。
- 动力学层：期望力矩是否超过关节连续力矩或峰值力矩。
- 伺服层：位置环、速度环、电流环带宽是否足够，限幅后是否产生积分饱和。
- 电机层：电压、电流、反电动势和热限制是否允许当前工况持续运行。
- 通信与实时层：控制周期、延迟和时间同步是否破坏闭环相位裕度。

## 第一阶段学习目标

第一阶段只做四件事：

1. 建立 DC 电机的电气和机械模型。
2. 用 PID 做单电机位置控制，观察阶跃响应和负载扰动。
3. 用级联控制拆分位置环、速度环和电流环。
4. 用 PMSM dq 模型理解 FOC，重点建立“`Iq` 近似对应力矩”的直觉。

这四件事足够支撑你继续学习机器人单关节控制、重力补偿、摩擦补偿、阻抗控制和驱动器工程约束。

## 建议实验方法

每个示例都生成 CSV。不要只看最终位置，要至少画这些曲线：

- 位置：是否超调、振荡、稳态误差。
- 速度：是否出现过大的峰值或高频震荡。
- 电流或力矩：是否频繁碰到限幅。
- 电压：是否被母线电压限制。
- 负载扰动：扰动进入后恢复速度和稳态误差如何变化。

如果你只做一个习惯，建议每次改参数后都问自己三个问题：

1. 响应变快，是不是靠更大的电流或电压换来的？
2. 响应变稳，是不是牺牲了跟踪带宽？
3. 如果这条轨迹放到机械臂上，会不会触发关节力矩或驱动器限幅？
