# 05 控制理论与调参工具

本章不是重新讲一遍控制理论，而是把机械臂关节调试时最常用的工具串起来：阶跃响应、频响、系统辨识、前馈和扰动观测器。

## 学习目标

学完本章后，你应该能判断：

- 阶跃响应里的超调、调节时间和稳态误差分别说明什么问题。
- 频响里的带宽和相位滞后如何限制闭环增益。
- 为什么系统辨识能帮助你估计惯量、阻尼和摩擦。
- 前馈和扰动观测器分别适合补偿哪类误差。

## 阶跃响应

阶跃响应是最直接的闭环测试：

```text
目标位置突然变化 -> 观察实际位置、速度、电流/力矩和控制输出
```

常见指标：

- 上升时间：系统响应速度。
- 超调：阻尼是否不足。
- 调节时间：多久稳定到目标附近。
- 稳态误差：是否存在负载、摩擦或模型偏差。

但机械臂真实轨迹通常不是硬阶跃，所以阶跃测试只适合暴露闭环特性，不代表最终轨迹跟踪质量。

## 频响和带宽

频响测试用不同频率的正弦输入激励闭环，观察输出幅值和相位：

```text
低频：输出几乎跟上输入
接近带宽：幅值衰减，相位明显滞后
高频：输出跟不上，噪声和延迟更明显
```

示例 [examples/sim_frequency_response.cpp](../examples/sim_frequency_response.cpp) 用一阶闭环近似演示带宽效应，输出 [outputs/frequency_response.csv](../outputs/frequency_response.csv)。

## 系统辨识

对单关节，可以从近似模型开始：

```text
tau = J * ddq + b * dq + tau_friction + tau_gravity
```

通过扫频、阶跃、慢速往返和负载扰动测试，可以估计：

- 等效惯量 `J`
- 粘性阻尼 `b`
- 库仑摩擦
- 重力项幅值
- 控制延迟

辨识结果不一定要非常精确，但能帮助你决定前馈项、限幅和轨迹约束是否合理。

## 前馈控制

反馈控制根据误差修正，前馈根据模型提前施加控制量：

```text
tau_cmd = tau_model(q, dq, ddq) + tau_feedback
```

对规划算法来说，前馈尤其重要。轨迹中的加速度、速度和重力项如果完全交给反馈环处理，关节会更容易出现滞后和误差尖峰。

## 扰动观测器

扰动观测器 DOB 用模型估计“命令力矩和实际运动之间解释不了的部分”：

```text
tau_dist ≈ tau_cmd - J * measured_ddq - b * measured_dq
```

示例 [examples/sim_disturbance_observer.cpp](../examples/sim_disturbance_observer.cpp) 比较负载扰动进入后，无 DOB 和有 DOB 的位置恢复。

DOB 的关键权衡是滤波带宽：

- 带宽高：扰动估计快，但容易放大噪声。
- 带宽低：更平滑，但补偿滞后。

## 和机械臂规划的关系

规划轨迹越接近关节带宽和力矩限制，反馈控制越难独自承担。好的规控落地通常依赖：

- 轨迹平滑，避免高频力矩需求。
- 有动力学前馈，降低跟踪误差。
- 有合理限幅，避免驱动器饱和。
- 有测试指标，能区分模型误差、控制带宽不足和传感器噪声。

本章示例给你两个工具：频响用于理解带宽，DOB 用于理解扰动补偿。
