# 06 机器人单关节控制

本章从电机侧走到机器人关节侧。机械臂关节不是裸电机，通常还包含减速器、负载惯量、摩擦、重力项、柔性和间隙。规划算法看到的关节力矩，也需要经过这些非理想环节才能落到真实运动上。

## 学习目标

学完本章后，你应该能判断：

- 电机侧编码器和负载侧编码器为什么可能不一致。
- 减速器、负载惯量和摩擦如何改变关节响应。
- 重力补偿和摩擦补偿为什么能显著降低跟踪误差。
- 阻抗控制和导纳控制分别适合什么接口和场景。

## 单关节模型

本学习包中的单关节模型在 [include/motor_control/robot/SingleJoint.hpp](../include/motor_control/robot/SingleJoint.hpp)：

```text
J * ddq = tau_cmd - tau_gravity(q) - tau_friction(dq) - tau_external
```

其中：

- `J`：负载侧等效惯量。
- `tau_gravity`：重力项。
- `tau_friction`：粘性摩擦和库仑摩擦。
- `tau_external`：外部扰动力矩或接触力矩。

这个模型不追求完整机械结构细节，而是保留调试关节控制最常见的误差来源。

## 重力和摩擦补偿

反馈控制可以修正误差，但如果重力和摩擦长期存在，反馈环会一直承担静态负载。加入前馈后：

```text
tau_cmd = tau_feedback + tau_gravity(q_des) + tau_friction(dq_des)
```

优势：

- 位置误差更小。
- 积分项压力更小。
- 慢速轨迹更平顺。
- 关节在某些姿态下不再明显下垂。

示例 [examples/sim_single_joint_gravity_friction.cpp](../examples/sim_single_joint_gravity_friction.cpp) 比较了只有反馈和加入重力/摩擦前馈的轨迹跟踪。

## 阻抗控制

阻抗控制把关节控制成虚拟弹簧阻尼：

```text
tau = K * (q_des - q) + D * (dq_des - dq) + tau_ff
```

刚度 `K` 越大，位置保持越硬；阻尼 `D` 越大，振荡越少但响应可能更钝。示例 [examples/sim_joint_impedance.cpp](../examples/sim_joint_impedance.cpp) 比较软阻抗和硬阻抗在外部扰动下的位移。

## 导纳控制

导纳控制通常在外部力传感器更可靠时使用：

```text
外力 -> 虚拟质量/阻尼/弹簧 -> 生成位置或速度指令
```

阻抗控制更像“给力矩接口设定力学行为”，导纳控制更像“根据外力生成运动指令”。如果底层驱动器只提供位置接口，导纳控制更容易落地；如果底层有高质量力矩接口，阻抗控制更直接。

## 和机械臂规划的关系

轨迹规划如果忽略重力、摩擦和关节负载变化，会把大量补偿压力交给反馈环。实际表现可能是：

- 上下运动误差不对称。
- 低速爬行时 stick-slip。
- 末端接触时刚度和阻尼不符合预期。
- 同一条轨迹在不同姿态下跟踪误差不同。

对规控算法工程师来说，关键是把轨迹、动力学前馈、关节补偿和底层限幅一起看，而不是只看几何路径。
